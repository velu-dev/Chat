# README
This is a Chat application using ActionCable rails

# Version

 * ruby 2.1.0 and above
 * rails 5.0 and above
# Step to start
  
  * git pull https://github.com/velutamil/Chat.git
  * cd Chat
  * bundle install
  * rake db:create db:migrate
  * rails s
